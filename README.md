# Dashboard de Inversiones

Plataforma web para la gestión y visualización de inversiones financieras.

## Características Principales

- **Dashboard**: Panel principal con resumen de inversiones y métricas clave
- **Reportes**: Visualizaciones interactivas de rendimiento, transacciones, análisis de activos y proyecciones
- **Configuración**: Gestión de perfil, métodos de pago, seguridad y preferencias
- **Multilingüe**: Soporte para español e inglés
- **Autenticación**: Sistema de autenticación seguro

## Tecnologías

- Astro
- Tailwind CSS v4
- JavaScript
- Go

## Estructura

El proyecto sigue una arquitectura modular con componentes reutilizables y páginas específicas para cada sección principal.

### Directorios Principales

- **backend/**: Contiene los microservicios del proyecto
  - Servicio de autenticación desarrollado en Go
  - Futuros microservicios

- **frontend/**: Contiene la landing page del sitio
  - Presentación del servicio
  - Información para usuarios potenciales

- **dashboard/**: Prototipo del dashboard de usuario
  - Panel de control con métricas de inversiones
  - Reportes interactivos (rendimiento, transacciones, análisis, proyecciones)
  - Configuración de perfil y preferencias
